 유형 문자열 폭발

# 코드 - Python

```python
import sys
input = sys.stdin.readline

string = input().rstrip()
fstring = input().rstrip()

while fstring in string:
    string = ''.join(string.split(fstring))
if string: print(string)
else: print("FRULA")


```

# 풀이

스택을 이용하지 않아서 시간초과를 받았다.
